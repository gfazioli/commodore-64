# RENAME

```basic
100 INPUT "NEW NAME"; A$
110 INPUT "PREVIOUS NAME"; B$
120 OPEN 15,8,15
130 PRINT #15,"R:"A$"="B$
140 CLOSE 15
```

# SCRATCH A FLOPPY

```basic
100 INPUT "NAME"; A$
120 OPEN 15,8,15
130 PRINT #15,"S:"A$
140 CLOSE 15
```

# FORMAT A FLOPPY

```basic
100 INPUT "NAME"; A$
110 INPUT "ID"; B$
120 OPEN 15,8,15
130 PRINT #15,"N:"+A$+","+B$
140 CLOSE 15
```

# GET THE INITIAL SYS

```basic
LOAD "FILENAME",8,1
PRINT PEEK(45) + 256*PEEK(46)
```

# AUTOMATIC LOAD AND RUN

```basic
LOAD "FILENAME",8 : REM PRESS SHIFT+RUN/STOP
LOAD "FILENAME",1,1 : REM PRESS SHIFT+RUN/STOP
```
